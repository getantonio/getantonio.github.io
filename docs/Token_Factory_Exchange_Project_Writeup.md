# Token Factory & Exchange Project Write-Up

Last updated: 2026-06-30

## Status

Local / in-progress prototype. The websites are not currently publicly accessible, so this should not be presented as a live production product.

## Honest Resume Positioning

This project can be used as self-directed project experience, not professional experience.

Recommended wording:

"Built a local crypto product prototype focused on token creation and exchange workflows. Used the project to learn how token products, user flows, trading/exchange concepts, and platform trust issues fit together. Public demo is not currently available; screenshots and walkthrough available."

## Interview Talking Points

- I do not have formal professional experience yet, so I built this to get practical exposure instead of only reading about crypto.
- The project helped me understand token creation, exchange flow, user trust, transaction clarity, and why operations/compliance matter in fintech.
- I used AI-assisted development, but I stayed involved in product decisions, testing, debugging, and understanding the workflow.
- The project is not live right now, so I am careful not to oversell it as a deployed product.

## What Not To Claim

- Do not claim it is live if it is not.
- Do not claim users, revenue, production traffic, audits, or institutional-grade security.
- Do not claim ownership of code you cannot explain.
- Do not use "exchange" in a way that suggests a regulated financial exchange or active trading platform.
